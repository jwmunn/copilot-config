# Session Learnings

This directory contains session learning artifacts produced by `/mslearn-session-learnings`.

Each file documents corrections, workarounds, and improvements identified during a coding session,
along with concrete patches that were suggested or applied to agents, prompts, skills, and instructions.

## Filename Pattern

`{date}-{description}-learnings.md`

Example: `2026-02-16-ssr-component-patterns-learnings.md`
